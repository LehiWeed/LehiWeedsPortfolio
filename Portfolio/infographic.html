5 Popular object oriented languages that have shaped my career

Reflection — Infographic

Lessons learned

This infographic taught me the value of concise visual communication. Choosing which languages to feature forced me to prioritize key facts and boil down complex ideas into clear, scannable points. I also improved my skills working with design tools and ensuring text legibility across different screen sizes.

What I like about this artifact

I like how the infographic provides an at-a-glance overview of languages that influenced my learning path. The visual hierarchy highlights strengths and use-cases for each language and makes the content approachable for readers who prefer visuals over long paragraphs.

What I would change

Given more time I'd tighten the color contrast for accessibility, add brief links to sample projects or code snippets for each language, and include a printable/alt text-friendly version for non-visual contexts.

How this helps in the future

Creating this infographic improved my ability to summarize technical writing, which I really enjoyed and needed.
